# Productivity of Factors of Production

## Project Overview

Interactive educational platform developed for the **National Interdisciplinary Contest "History and Society in a Virtual Dimension"**, Socio-Human Sciences section (Economics). The project facilitates the learning of fundamental concepts regarding productivity through theoretical modules, calculation tools, and automated evaluation systems.

---

## Project Structure

The application is organized into **seven core modules**:

### 1. Theoretical Module
Content structured on five directions:
- Definitions
- Forms of productivity (global/partial)
- Calculation formulas
- Influencing factors
- The law of diminishing returns

### 2. Didactic Diagrams
Graphical representations of economic processes and correlations between key indicators.

### 3. Interactive Calculator
Calculation engine for global, labor, capital, and land productivity, including marginal productivity, providing instant interpretation of results.

### 4. Worksheets
Interactive exercises featuring fill-in-the-blanks, multiple choice, and argumentation tasks with immediate feedback.

### 5. External Resources
Curated list of educational platforms (Khan Academy, MIT OpenCourseWare) and specialized academic resources.

### 6. Evaluation System
Three difficulty levels with server-side validation, integrated timer, and security measures to prevent client-side score manipulation.

### 7. Bibliography
Centralization of academic sources and official textbooks used in the development process.

## Contest-Specific Features

### Scientific Accuracy
Content strictly follows the terminology and mathematical formulas from the national economics curriculum.

### Server-Side Validation
Test results are processed via API to ensure the integrity of the evaluation process.

### Interactivity
Calculators allow real-time simulations, translating economic theory into practical application.

### Accessibility
Clean, distraction-free interface optimized for the learning process and user experience.
